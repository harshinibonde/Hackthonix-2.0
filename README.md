# StartHub 🚀
### Discover the Next Big Startup

StartHub is a startup discovery platform that helps emerging startups showcase their ideas and allows users to explore innovative projects in an engaging way.

Inspired by platforms like Product Hunt and Netflix-style browsing interfaces, StartHub provides a modern environment where founders can gain visibility and users can easily discover new startups.

---

## Features

### Startup Discovery

Browse innovative startups through categorized sections such as:

- Trending Startups
- Recently Launched
- AI Startups
- SaaS Tools
- Student Startups
- Sustainability Startups

### Submit Your Startup

Founders can submit their startups by providing:

- Startup name
- Tagline
- Description
- Founder details
- Website link
- Logo and banner images

### Upvote System

Users can upvote startups they find interesting, helping the best projects gain visibility.

### Brand of the Week

Each week the platform highlights a Brand of the Week based on:

- Number of upvotes
- Recent activity
- Community engagement

### Startup Profile Pages

Each startup has a dedicated page displaying:

- Startup description
- Founder story
- Product screenshots
- Website links
- Category information

Backend:
- Next.js API Routes

Database:
- PostgreSQL
- Prisma ORM

Authentication:
- NextAuth (Google & GitHub login)

---

## How It Works

1. Founders submit their startup through the Submit Startup form.
2. The startup information is stored in the database.
3. Startups appear in the discovery feed.
4. Users browse and explore startups.
5. Users can upvote startups.
6. The most engaging startup becomes Brand of the Week.

---

## System Architecture

User → Frontend (Next.js) → API Routes → Database (PostgreSQL)
